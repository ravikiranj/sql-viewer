# sql-viewer
SQL Viewer Chrome extension

## Setup
* Install `nodejs` and `npm`
* Install dev dependencies via `npm install`
* Run `grunt`

## Dev Deployment
* Open `chrome://extensions` in Google Chrome
* Load unpackaged extensions and point to `src/` folder
